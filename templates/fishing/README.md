# Fishing Project

Create a relaxing fishing simulator with realistic water physics and nature ambience.

## Hardware
- ESP32
- OLED Display (128x64 I2C)
- Potentiometer
- Vibration Motor

## Libraries
- Adafruit_SSD1306
- Adafruit_GFX

## Goals
1. Casting mechanic with power meter
2. Procedural water animation
3. Fish AI (biting, fighting)
4. Tension-based reeling
5. Day/night cycle
6. Catch log and stats
