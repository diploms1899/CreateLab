# Platformer Project

Build a side-scrolling platformer game with pixel-art graphics, enemies, power-ups, and retro audio.

## Hardware
- ESP32
- OLED Display (128x64 I2C)
- 3x Push Buttons
- Buzzer

## Libraries
- Adafruit_SSD1306
- Adafruit_GFX

## Goals
1. Character movement (left/right/jump)
2. Jump physics with gravity
3. Enemy AI behavior
4. Score and lives system
5. Sound effects via buzzer
6. Level progression
