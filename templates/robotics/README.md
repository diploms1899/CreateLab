# Robotics Project

Design and program a custom autonomous robot with sensor fusion and PID control.

## Hardware
- ESP32
- OLED Display (128x64 I2C)
- Ultrasonic Sensor (HC-SR04)
- 2x IR Line Sensors
- IMU (MPU6050)
- 2x DC Motors + L298N Driver
- Servo (SG90)

## Libraries
- Adafruit_SSD1306, Adafruit_GFX
- Adafruit_MPU6050

## Goals
1. Motor control with PWM
2. Obstacle avoidance
3. Line following
4. PID speed control
5. Sensor fusion (IMU + encoders)
6. Autonomous navigation
