# Calculator Project

Engineer a full scientific calculator with expression parsing and a clean user interface.

## Hardware
- ESP32
- LCD 16x2 (I2C)
- 4x4 Matrix Keypad

## Libraries
- LiquidCrystal_I2C
- Keypad

## Goals
1. Basic arithmetic (+, -, *, /)
2. Expression parsing (shunting-yard)
3. Scientific functions (sin, cos, sqrt)
4. Memory store/recall
5. History navigation
6. Error handling (div by zero, overflow)
